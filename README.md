### nemo-util

A plugin ðŸ”Œ for [Nemo](http://nemo.js.org) automation framework that provides handy utilities for browser automation.

##### Installation

```console
npm install nemo-util -D
```
or
```console
yarn add nemo-util --dev
```

##### Configuration

Add `nemo-util` plugin under `plugins` section of your `config/config.json` or `nemo.config.js` file.

```javascript
    	"plugins": {
		    "util": {
		        "module": "nemo-util"
		     }
		 }
```
